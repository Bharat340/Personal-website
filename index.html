Latest Projects

WTNIC

Web Text based Network Industry Classification

New

Our goal is to capture organization form, competition, and industry change through Text-mining of private and public firm webpages. In this project, I've implemented feature selections after preprocessing of data, TF-IDF, Jaccard, MinhashLSH, Doc2Vec (document embedding using neural networks) on a dataset consisting of 807K companies’ web crawled (11 TB) data for last two decades.

Merchant Integration Platform

We developed generic RESTful APIs for PayU, PayUmoney & Citrus to simplify integration process for merchants with high volume of transactions. For this we used, Spring MVC framework, Kong framework, and generic table structures from Schema.org

AB Testing Framework

I've designed the framework for backend and developed some generic features for multiple campaigns, generating complex reports for each test for later analysis. For this I used, Spring MVC framework, Hibernate, MySQL, Nginx.

PayUmoney Authentication System

Previously we were using client based authentication. We wanted to implement a stable authentication system for all of our stake holders (Customers, Merchants, Admins, and other staff). So we developed an authentication system using oAuth2 concept with Spring Security. And generalized our URLs accordingly to manage ACL automatically.

Message Broker Framework

Earlier, most of our processes were handling parallelization through multiple thread forks or managed manually by the developers in different use cases. But, after we explored the potenial use of Message Queues, we built a generic platform which can efficiently manage all asynchronous processes. This architecture was initially built with RabbitMQ but later on we changed it to Kafka. I was a part of the backend development team for this.
